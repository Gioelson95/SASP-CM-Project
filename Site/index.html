Motion Controlled Guitar Effects

Learn More

Project focus

Focus of the project is to simulate some guitar effect and control them by the movement of the body tracked by some sensors like Micrisoft Kinect and smatphone ones.



Sofware used

Effects are written using Supercollider and controlled by OSC message passing throught Processing. To make the guitar sound more realistic have been used the Logic Pro X amp simulation with the help of Soundflower to connect the audio bus from Supercollider to Logic Pro X.



Possible developments

Future imlpementations in order to improve the project could be a more detailed skeleton recognition using Microsoft Kinect and a greather movement recognition using machine learning.
